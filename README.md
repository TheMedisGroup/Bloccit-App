## Bloccit: a Reddit replica to teach the fundamentals of web development and Rails.

## Installation
Clone the repo:
```
git clone git@github.com:mikeMedis/bloccit.git
```
Run:
```
bundle install
```
Website can be found at:
https://git.heroku.com/glacial-woodland-7335.git
